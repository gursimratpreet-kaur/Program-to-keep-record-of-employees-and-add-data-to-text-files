# Program-to-keep-record-of-employees-and-add-data-to-text-files
Interactive program that displays four options.
1. To display managers- It displays information about the employee in the form of table.
2. Add manager- To add a new manager. It asks for ID, name, city, days worked and adds the manager.
3. Find manager- It is used to look for a particular manager and displays information about it using an ID.
4. End program- This adds all the data to two different text files.
